# BioGenie - AI Instagram Bio Generator

BioGenie helps you create unique and engaging Instagram bios using AI. Simply provide some keywords, desired tone, and emoji preferences, and let our system generate multiple creative options for you. Optimize your Instagram presence effortlessly!

**Features:**
- AI-powered bio generation.
- Customizable tone and emoji preferences.
- Multiple bio options to choose from.

**How to use:**
1. Visit the web application.
2. Enter your desired keywords, select a tone, and add emoji preferences.
3. Click 'Generate My Bio!'
4. Review the generated options and select your favorite.

This project is built using Next.js (or similar) and integrates with OpenAI's GPT-3.5 Turbo API for content generation.
