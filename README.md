# BioBoost AI

BioBoost AI is an AI-powered web application that generates creative and personalized Instagram bios. Users input their industry, hobbies, and desired tone, and the app provides multiple bio options. Perfect for influencers, businesses, or anyone needing a fresh online presence.

## Features
- AI-driven bio generation
- Customizable inputs
- Multiple bio options
